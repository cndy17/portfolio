A blockchain-backed professor evaluation system giving students a tamper-proof platform for honest feedback. Smart contracts prevent review manipulation — no admin can delete them. JavaScript frontend, SQLite backend, Solidity contracts.

## The problem

Existing professor review platforms can be gamed — reviews deleted, ratings inflated, content moderated by institutions with conflicts of interest.

## The solution

Smart contracts make reviews immutable once submitted. One student, one review per course. The web3 complexity is invisible to the end user — it looks like any other review site.
