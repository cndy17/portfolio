A desktop music player built in Java. Artists upload tracks; listeners browse, play, and create playlists. Built with object-oriented design principles and a custom Java Swing GUI.

## What I learned

This project made OOP concrete. Inheritance, polymorphism, and encapsulation stopped being textbook concepts and became real design decisions with trade-offs.
